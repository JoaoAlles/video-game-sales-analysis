# Video Game Sales Analysis

Data analysis project using Python and Pandas to explore historical trends in the video game market.

## Objective

The objective of this project was to practice initial data analysis concepts using a public video game sales dataset.

During the analysis, I aimed to answer questions related to market behavior over the years, platforms with the highest sales volume, and the best-selling games in history.

---

## Dataset Used

Public dataset available on Kaggle:

* Video Game Sales Dataset

The dataset contains information about:

* Game names
* Platforms
* Release years
* Genres
* Publishers
* Global and regional sales

---

## Technologies Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

---

## Questions Answered

* What were the best-selling games in history?
* Which platform sold the most globally?
* Which platform had the highest sales each year?
* What was the best-selling game of each year?
* Which year had the highest sales volume?
* Which platform released the most games?
* Which platform had the highest global sales?

---

## Some Insights Found

* PS2 appears among the platforms with the highest global sales volume.
* The video game market experienced accelerated growth during the 2000s.
* Some platforms had very strong sales peaks during specific periods, such as the Wii.
* Certain franchises dominated sales for several consecutive years.
